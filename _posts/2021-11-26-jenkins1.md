# 1. 개요
Tomcat 에 Jenkins 설치
Jenkins 는 기본적으로 jetty 로 구성되어 패키지 제공되고 있으나,
war를 Tomcat에 배포하는 것으로 설치해보기로 한다.

# 2. 설치
## 2.1 다운로드
[여기 클릭](https://www.jenkins.io/download/)

" 여기는 그림 "
위 LTS 항목을 다운로드 받았다.

## 2.2 Tomcat 환경변수 설정
JEKINS_HOME만 세팅하였다.

```bash
export JENKINS_HOME="your path"
```

## 2.3 war를 배포
```bash
/app/servers/tomcat1/webapps/jenkins.war
```
여기까지가 아주 심플하게 설치는 완료되었다.
